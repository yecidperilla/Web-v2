Solución de Problemas Comunes

Error 0xC004F074 - No se pudo contactar el servidor KMS

1

Verifica tu conexión a internet

2

Prueba con un servidor KMS alternativo:

# Ejecutar en PowerShell como Administrador
slmgr /skms kms8.msguides.com
slmgr /ato

Office no se activa (Error 0x80070005)

1

Reparar permisos de Office:

# Ejecutar en PowerShell como Administrador
icacls "%ProgramFiles%\Microsoft Office\Office16\ospp.vbs" /grant *S-1-15-2-1:RX
2

Reparar instalación de Office desde Panel de Control > Programas

Métodos de Activación

1. HWID (Hardware ID)

Para: Windows 10/11

Duración: Permanente (vinculado a tu hardware)

Ventajas: No requiere renovación, sobrevive a reinstalaciones

2. Ohook

Para: Office 2013-2021

Duración: Permanente

Ventajas: No modifica archivos de Office, fácil de revertir

3. KMS

Para: Windows y Office

Duración: 180 días (se renueva automáticamente)

Ventajas: Funciona en todas las versiones

4. KMS38

Para: Windows 10/11 Enterprise LTSC

Duración: Hasta 2038

Ventajas: Casi permanente para versiones Enterprise

Solución de Problemas

Error de Activación 0xC004F074

1

Verifica tu conexión a internet

2

Prueba con diferentes servidores KMS:

# Servidores KMS alternativos:
kms8.msguides.com
kms.digiboy.ir
kms.ddns.net
3

Verifica que el firewall no bloquee el puerto 1688

Office no se activa (Error 0x80070005)

1

Reparar permisos de Office:

# Ejecutar en PowerShell como Administrador
$officePath = "$env:ProgramFiles\Microsoft Office\Office16"
if (Test-Path $officePath) {
    icacls "$officePath\ospp.vbs" /grant *S-1-15-2-1:RX
}
2

Reparar instalación de Office desde Panel de Control

Problemas de Activación después de actualización

1

Reactivar usando el mismo método original

2

Si persiste, usar el comando de rearranque:

# Para Windows:
slmgr /rearm

# Para Office:
cd "%ProgramFiles%\Microsoft Office\Office16"
cscript ospp.vbs /rearm

Activación HWID para Windows

1

Requisitos:

  • Windows 10/11 Pro, Enterprise o Education
  • Conexión a internet para la activación inicial
2

Ejecutar el comando de activación:

# Ejecutar en PowerShell como Administrador
irm https://massgrave.dev/hwid | iex
3

Confirmar activación:

# Verificar estado de activación
slmgr /xpr

1

Requisitos:

  • Office 2013-2021 instalado
  • Versiones compatibles: ProPlus, Professional, Standard
2

Ejecutar el comando de activación:

# Ejecutar en PowerShell como Administrador
irm https://massgrave.dev/ohook | iex
3

Verificar activación:

Abre cualquier aplicación de Office y verifica que no aparezcan mensajes de activación.
